基础用法
基本用法，可以使用value实现数据的双向绑定。 可以设置class来改变输入框的宽度及其他样式，默认 100%。
输入框类型
除支持input输入框的基本类型，还支持设置
type为integer(整数)、positiveinteger(正整数)、positivenumber(正数)。
带Icon的输入框
通过iconClass属性添加类(例如fa图标集)可以在输入框右边加一个图标。 点击图标，会触发onIconClick事件。
禁用状态
通过添加disabled属性可设置为不可用状态。
所有事件
点击图标时触发onIconClick事件，输入时触发onInput事件，获得焦点时触发onFocus事件，
失去焦点时触发onBlur事件，回车触发onSubmit事件。
必填样式
输入框必填样式，有值后会恢复为正常样式. 2.10.16开始required支持ko对象，可以动态设置必填样式
