ycloud (俗称why cloud或云采cloud) 是友云采前端团队常年基于企业级应用页面总结出品的一套用于构建用户界面的UI框架/UE规范。 我们在基于knockoutjs的基础上，为您提供了一系列丰富可用的组件，方便您进行快速搭建一套企业级管理应用界面。
支持knockoutjs-3.4.2版本以及knockoutjs-3.5.0-rc版本（默认设置ko.options.createChildContextWithAs = true）

NPM 安装

推荐使用 yarn(或npm) 来安装，享受生态圈和工具带来的便利，更好地和 webpack 配合使用，当然，我们也推荐使用 ES2015。
