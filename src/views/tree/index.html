基础用法
最简单的用法，点击可获取选中的值，树节点数据格式:{id:'', name:'', children: []}
异步加载
使用 loadData参数，传入获取子节点的函数即可
当前选中的是：
带checkbox多选
使用 multiple:true参数，即可显示checkbox
传入默认勾选值可使用 selectedId 根据数据的id字段判断选中
selectedItem 可返回选中的项的全部数据内容
鼠标移入可编辑
使用 extraText:'编辑',extraEdit:$root.extraEdit参数
传入编辑按钮显示文本，以及点击编辑回调事件，仅当鼠标移入当前及选中当前节点时起作用
节点可配置项checkbox+图标
使用 nodeOption:nodeOption:参数
nodeOption: {
showCheck (nodeData) // 节点是否显示checkbox的判断函数
nodeIcon (nodeData) // 节点图标样式 根据nodeData内容判断
}
