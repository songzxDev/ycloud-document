基本用法
基本用法。dataList绑定可选择的数据，value绑定已选择的数据，onChange可绑定自定义事件
(当点击item时，已选数据发生变化时触发)。
多选
通过设置属性multiple可以开启多选模式。多选模式下，multiValue接受数组类型的数据，所返回的也是数组。
默认值
单选模式设置属性value初始化默认值(基本格式：例{value: 1, label: '北京'})，
多选模式设置属性multiValue初始化默认值(基本格式：例{value: 1, label: '北京'}, {value: 6, label: '北平'})。
可清空
通过设置属性clearable可以清空已选项。
可搜索
通过设置属性filterable可以开启搜索模式。仅适用于单选模式。
自定义内容
自定义内容
通过设置属性hasmore可以开启更多模式，onmore用于绑定点击更多触发的事件。
